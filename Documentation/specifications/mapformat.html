map format

Map files are stored in GameServer/res/maps/ and are loaded when the server starts.
Map files are ASCII files formatted as follow: (based on the AIChallenge specifications) 
players %BOT_COUNT%
cols %COLS%
rows %ROWS%
m [.%*0-9]{%ROWS}
The following tokens are allowed: 
players		number of bots the map is designed for [2-10]
cols		map column count
rows		map row count
m			map row descriptor
You should add a " " (space) after a token.
After a row descriptor token, each character represents a cell in the map row: 
.			void
%			wall
*			food spawn
0-9			ant hill for bot i
Note that you can define several hills for the same bot.
