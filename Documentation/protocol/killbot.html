killbot

Remove a bot on the game server.

Client request

{
	"type": "killbot",
	"content": {
		"token": "21014616ea7d673e04046a132b6593ad",
	}
}

Server response

{
	"type": "killbot",
	"error": 0,
	"message": "Bot removed",
}

Errors

ID message
101 Token does not exist
102 Bot is connected
