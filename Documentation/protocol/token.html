Create a bot on the game server.

Client request

{
	type: "token",
	content: {
		nick: "mybot"
	}
}

Server response

{
	type: "token",
	error: 0,
	message: "Bot created",
	content: {
		token: "21014616ea7d673e04046a132b6593ad"
	}
}

Errors

content: null
