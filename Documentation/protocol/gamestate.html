gamestate

Warning: work in progress!
Communication during one game round.
Once you received a "gamestate" message, you have a certain amount of time to send back your actions in a "gameactions" messages otherwise you will receive a "gamemute" message meaning you cannot play in this game anymore.

Server info

{
	"type": "gamestate",
	"error": 0,
	"message": "Current state of the game, please give your actions"
	"content": {
		"gameobjects": [
			{
				"type": "wall",
				"col": 45,
				"row:" 24
			}
		]
	}
}

Warning

"content": depends on the type of game implemented in the game server via the Game class.
