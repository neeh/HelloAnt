ranking

Gets the list of bots ordered by score in a descending order.

Client request

{
	"type": "ranking",
	"content": {}
}

Server response

{
	"type": "ranking",
	"error": 0,
	"message": "4 bots are ranked",
	"content": {
		"ranking": [
			"lola51",
			"BlackBerry",
			"testr",
			"jacky"
		]
	}
}
