gamestart

Informs the bot that it was put in game.

Server info

{
	"type": "gamestart",
	"error": 0,
	"message": "A game just started"
	"content": {
		"map": {
			"cols": 400,
			"rows": 300,
			"hills": [
				{
					"col": 250,
					"row": 116
				}
			]
		}
	}
}

Warning

"content": depends on the type of game implemented in the game server via the Game class.
