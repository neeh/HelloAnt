setmode

Change the gaming mode of a bot.

Client request

{
	"type": "setmode",
	"content": {
		"mode": "regular"
	}
}

Server response

{
	"type": "setmode",
	"error": 0,
	"message": "Mode changed to regular,
}

Errors

ID message
101 Unknown mode %MODE%
102 Already in regular/training mode
