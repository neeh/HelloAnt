connect

Connect a bot on the game server.

Client request

{
	"type": "connect",
	"content": {
		"token": "21014616ea7d673e04046a132b6593ad",
		"mode": "training"
	}
}

Server response

{
	"type": "connect",
	"error": 0,
	"message": "Connected in training mode",
	"content": {
		"nick": "mybot",
		"score": 1200
	}
}

Errors

"content":

ID message
101 Token does not exist
102 Bot with this token is already connected
103 Already connected with bot %NICK%
