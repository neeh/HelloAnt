login

Login with a bot on the game server.

Client request

{
	"type": "login",
	"content": {
		"token": "21014616ea7d673e04046a132b6593ad",
		"mode": "training"
	}
}

Server response

{
	"type": "login",
	"error": 0,
	"message": "Logged in (training mode)",
	"content": {
		"nick": "mybot",
		"score": 1200
	}
}

Errors

"content":

ID message
101 Token does not exist
102 The bot with this token is already logged in
103 Already logged in with bot %NICK%
